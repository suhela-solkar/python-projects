# Python Projects

This repository contains beginner-level Python projects created to practice programming fundamentals and logic building.

## Projects Included
- Number Guessing Game – An interactive game using loops, conditionals, and the random module
- Password Strength Checker – A program to evaluate password strength using string operations and validation

## How to Run
Make sure Python is installed, then run:
python number_guessing_game.py 
python password_strength_checker.py
