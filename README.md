# Python Projects

This repository contains beginner-level Python projects created to practice programming fundamentals and logic building.

## Projects Included
# Number Guessing Game

A simple Python game where the user guesses a randomly generated number.

## Concepts Used
- Random module
- Loops
- Conditional statements
- User input handling

## How It Works
The program generates a random number, and the user keeps guessing until
they find the correct number.

# Password Strength Checker

A Python program that checks the strength of a password based on
length and character variety.

## Concepts Used
- Strings
- Conditional logic
- Functions

## Features
- Checks minimum password length
- Verifies presence of uppercase, lowercase, digits, and symbols

## How to Run
Make sure Python is installed, then run:
python number_guessing_game.py 
python password_strength_checker.py
